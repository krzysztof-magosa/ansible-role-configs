## Description
Generic configs templater

## Requirements
* Ansible 2.8+ (tested on 2.8, 2.9 but likely works on older versions as well)

## Supported systems
* CentOS (tested on 7, 8)
* Debian (tested on 10 buster, 11 bullseye)
* Ubuntu (tested on 18.04 bionic)
